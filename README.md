## Current Projects

### Token Counter CLI (Week 1–2 Assignment)

A Python command-line tool that:

* Reads text files
* Counts characters, words, and tokens
* Uses the `tiktoken` library for OpenAI-compatible tokenization
* Estimates OpenAI API input costs based on selected models
* Handles invalid file names, empty files, and unsupported models

### Features

* Character count
* Word count
* Token count
* OpenAI model selection
* Input cost estimation
* Error handling
* Virtual environment support
