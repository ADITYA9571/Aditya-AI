# Aditya-AI

Building and understanding agentic AI systems.

This repository documents my journey in Artificial Intelligence through projects, experiments, certifications, training programs, internship assignments, and hands-on implementations.

## Current Projects

### Token Counter CLI (Week 1–2 Assignment)

A Python command-line tool that:

* ReCounts characters, words, and tokens from text files
* Estimates OpenAI API input costs based on selected models using tiktoken library

### FastAPI + Docker (Week 3 Assignment)

A containerized FastAPI application that:

* Provides REST API endpoints for token and cost analysis
* Runs inside Docker for consistent deployment and execution across environments

### Features

* Character, word, and token counting using `tiktoken`
* OpenAI API cost estimation based on selected models
* REST API development using FastAPI
* Request validation using Pydantic models
* Async endpoint implementation
* Docker containerization for consistent deployment
