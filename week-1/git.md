# Week 1 Git Commands

## Initialize new repo

git init 

## Check Repository Status
git status

## View Current Branch
git branch

## View Branch Tracking Information
git branch -vv

## Create a New Branch
git checkout -b week-1

## Switch Branches
git checkout main
git checkout week-1

## Add Files
git add .

## Commit Changes
git commit -m "Week 1 Python basics"

## Push Current Branch
git push

## Push New Branch and Set Upstream
git push -u origin week-1

## Remove Virtual Environment from Git Tracking
git rm -r --cached aadi_env

## Merge Branch into Main
git checkout main
git merge week-1

## View Repository Files
ls

## Move to Parent Directory
cd ..

## Move into Week 1 Folder
cd week-1
